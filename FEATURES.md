# How to Use Yahli

## 🧠 Basic Interaction

Everything works through natural language.

Examples:


- "Remind me to call Shmuel"
- "Send a message to my brother"
- "What should I focus on today?"

---

## 📌 Tasks

You can:

- Add tasks with or without date
- Let Yahli decide the best timing
- Set priority (optional)

Example:
"Finish project presentation this week"

Yahli will:

- Choose optimal day
- Add it to your system
- Balance workload

---

## 🔁 Routines

Example:
"Workout every Sunday and Wednesday"

Tracks:

- Streaks
- Completion count

---

## 📲 Sending Messages

Example:
"Send a message to my brother that I’ll be late"

Yahli:

- Finds contact
- Writes message
- Sends it

---

## 🧩 Complex Tasks

Example:
"Replace client Shmuel with Hezi everywhere"

Yahli:

- Finds all references
- Updates system
- Ensures consistency

---

## 📅 Workday Planning

Example:
"I start at 10 today"

Yahli:

- Calculates work hours
- Schedules tasks
- Updates calendar

---

## 🤖 AI Partners

You can create assistants like:

- Project manager
- Reminder system
- Follow-up agent

They:

- Track progress
- Remind automatically
- Keep context

---

## 🔍 Smart Search

Example:
"Find trends in AI tools"

Yahli:

- Breaks into multiple queries
- Searches dozens of sources
- Returns structured insights

---

## 📊 Graphs

Example:
"Show me graphs of my tasks by category"

Yahli:

- Collects data
- Generates visualization

---

## ⚙️ Advanced (Optional)

- Import Google Sheets
- Create automation flows
- Build new data tables

---

## 💡 Pro Tip

You don’t need to be precise.

Yahli understands intent:

- “my brother”
- “that project”
- “something like last time”

---

## 🧠 Philosophy

Yahli is built to:

- Reduce thinking overhead
- Act proactively
- Manage complexity for you

---

Use it like a human assistant — not like a tool.
